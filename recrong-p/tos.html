第一部分：服务定义与性质

Recrong 仅是一个基于 Web 技术的任务调度与触发工具（Scheduler & Trigger Tool）。本平台提供的核心价值为：定时逻辑执行、触发流程管理以及任务触发结果的记录。

本平台不提供电信运营服务，不具备短信通道能力，不直接发送任何短信。 所有短信均由您所配置的第三方运营商或 API 服务商发送。

2. 独立缔约关系

本平台与您注册时配置的第三方短信 API 服务商（例如 Loading...）之间不存在任何关联关系、代理关系、合作关系或委托关系。

您与第三方服务商之间的：

  • 账户开通与身份校验
  • 费用结算与扣费争议
  • 发送成功率、延迟、失败率
  • 内容审核、风控策略、封号处理

均与本平台无关。

3. 用户独立责任

使用本平台意味着您完全理解并接受：所有任务触发行为均由您自行配置， 您需对自己的配置、内容、链路安全及第三方服务的费用承担全部责任。
